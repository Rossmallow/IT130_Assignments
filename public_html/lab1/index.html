  1. The difference between a web server and a web client is that a web server hosts websites, and a web client accesses websites.
  2. A web server is used to serve an HTML document to web clients that request it. If a document is just stored locally and not made publicly available like it would be on a server, clients will not be able to find it.
  3. The document type statement is required to let the web client know what kind of document it is reading. The file will be read differently depending on the type of document it is.
  4. Every HTML document needs a document type statement at the bare minimum. The meta tag with the charset attribute set to some value is recommended due to a common hack.
  5. The <meta> and <img> tags don't contain any content, and therefore don't need to be closed around anything.
  6. To subscribe to News postings in D2L, first click on the drop-down-arrow next to the desired news feed. Next, click on the "Notifications" option to be taken to a new page. Select which occasions to receive sms and/or email notifications.
